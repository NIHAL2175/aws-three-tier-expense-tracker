💰 AWS Serverless Expense Tracker
📌 Project Level: Intermediate
Serverless Expense Tracking web application built using AWS services like EC2, Lambda, API Gateway, DynamoDB, IAM Role and CORS with a scalable and cost-efficient architecture.

⚡ What This Project Does
Track your expenses in real-time, no servers, no hassle.

• Add, view, update and delete expenses
• Backend powered entirely by AWS serverless services
• Fast, scalable and cost-efficient

🧠 How It Works
Think of it like this:

User → API Gateway → Lambda → DynamoDB → Response → UI

• Frontend sends request
• API Gateway routes it
• Lambda processes logic
• DynamoDB stores data
• CloudWatch logs everything

🧰 Tech Stack
• EC2 Instance
• AWS Lambda
• API Gateway
• DynamoDB
• CloudWatch
• HTML, CSS, JavaScript
